# HOME DECOR WEB PROGRAMMING Project

## Project Overview
This project is a web-based e-commerce system that allows users to:
- Registration using TRN, phone number, and personal details.
- Login using TRN and password.
- Login system with attempt lockout after 3 failed tries.
- Password reset functionality
- Browse products and add them to a cart.
- Update quantities, remove items, view totals with tax & discounts
- Checkout and generate invoices.
- Track personal invoices.
- Search invoices by TRN
- View analytics like user distribution by gender and age.

The system uses **localStorage** to store data.

## How to Run the Project
Option A — Run Locally
1. download the project folder.
2. Open `index.html` in a browser (Chrome preferbly).
3. Use the navigation menu to:
   - Login / Register
   - Browse products (`products.html`)
   - Add products to your cart
   - Checkout and view invoices
   - Open `dashboard.html` to view charts.
   - Search invoices by TRN or view your own invoices.
